# Matchy-Grid
#### Matchy Grid is a playful, single‑page web app designed to help beginners, students, and aspiring entrepreneurs discover business ideas and take their first steps toward building them.
This is my first project I did for Code with AI Hackathon By Microsoft Asia. 
---
## 🚀 Features
- Interactive idea grid with curated business concepts
- Suggestive pathways: simple starter steps for each idea
- Favorites & progress tracking using localStorage
- Surprise Me button for random inspiration
- Clean, colorful design with pastel palette and modern fonts
